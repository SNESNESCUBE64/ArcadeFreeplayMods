# Stratovox Free Play
This is a mod to original Taito Licensed Stratovox ROMs that adds free play to the game. 

## Patch information
One patch are provided for the *stratvox* ROM set as found in MAME. It has been tested for this ROM set only and may not work on other revisions of Stratovox. The patches are designed to be used with LunarIPS. 


| **Patched ROM Name** | **Size** | **CRC-32 Checksum** | **IC Location** |
|----------------------|----------|---------------------|-----------------|
| ls01.bin             |    2k    |       13457EB8      |       4/5J      |
| ls06.bin             |    2k    |       BAC01655      |       11J       |
