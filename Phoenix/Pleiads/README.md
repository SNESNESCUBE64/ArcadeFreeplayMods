# Pleiades Freeplay
This is a freeplay mod for Pleiads. 

## Patch information
### Supported ROM Sets
| **ROM Set** | **MAME Working?** | **Machine Working?** |
|-------------|:-----------------:|:--------------------:|
| pleiads     |        Yes        |         Untested     |
| pleiadce    |        No         |         No           |
| pleiadsi    |        No         |         No           |
| pleiadsia   |        No         |         No           |
