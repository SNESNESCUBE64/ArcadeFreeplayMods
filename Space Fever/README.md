# Space Fever Freeplay
This is a mod for original Nintendo Space Fever ROMs that adds free play to the game.

## Patch information
Several patch files are provided for the *spacefev* ROM set as found in MAME. It has been tested for this ROM set only and may not work on other revisions of Space Fever. The patches are designed to be used with LunarIPS.
