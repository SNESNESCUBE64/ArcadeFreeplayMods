# Phoenix Freeplay
This is a freeplay mod for Amstar/Centuri Phoenix. So far only the Centuri sets are supported. There are an absurd amount of phoenix ROM sets, not all of them will be tested.

## Patch information
### Supported ROM Sets
| **ROM Set** | **MAME Working?** | **Machine Working?** |
|-------------|:-----------------:|:--------------------:|
| phoenix     |        Yes        |         Untested     |
| phoenixa    |        Yes        |         Yes          |
| phoenixb    |        Yes        |         Yes          |
